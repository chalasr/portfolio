This application requires:
-------------

- PHP > 5.4
- Composer
- NPM
- Gulp
- Compass
- CoffeeScript
- jQuery
- Materialize
- Bootstrap

Getting Started
---------------

 - Install vendor using composer

 ``` composer install ```

 - Create database

 ``` doctrine:database:create ```

 - Create schema of database

 ``` doctrine:schema:create ```

 - Start server

 ``` php app/console server:run ```

Enjoy !
